Text based elements

Strong:

It is semantically used to give strong importance to text, its the most popular option for bolding text.

 i:

It is used semantically to convey text in an alternative voice or tone

u:

It underlines text.

b:

It's used to draw the reader's attention to the text

Strike

This strikes text

small

This makes the text small.

em

This is an element that stresses text.
